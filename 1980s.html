The 1980s

  • 1981: The first IBM PC is released, using Microsoft's MS-DOS operating system. It used an Intel chip, two floppy disks, and an optional colour monitor.
  • 1983: Apple releases the Lisa, the first computer with a GUI. It featured drop-down menus and icons. It was not commercially successful, but it later evolved into the Macintosh.
  • 1983: The Gavilan SC is the first portable computer marketed as a "laptop".
  • 1985: Microsoft announces Windows, their response to Apple's GUI introduced with Lisa.
  • 1985: The first dot-com domain name is registered, by the Symbolics Computer Company.
