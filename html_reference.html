HTML Tags Reference

Tag Description Example
<!-- --> The comment tag. Any text in it won't affect the code. Useful for reminding yourself what a section of code does. <!--Info Table-->
<!DOCTYPE> Necessary to define document type. <!DOCTYPE html>
<head> Define the head section of the document, where things like the title usually are. <head>
</head>
<body> Used to define the body, where most of the content is stored. <body>
</body>
<h1>, <h2>, etc. Header tags. The number indicates the size, with larger numbers being smaller headers. <h5>Welcome</h5>
<p> The paragraph tag, pretty self-explanatory. <p>This is a paragraph</p>
<a> The hyperlink tag, use the href attribute to link to another page. <a href="http://www.google.ca">
<title> This tag sets a title for your page, which shows up in the browser tab. <title>Kyle's Website</title>
<img> Used to display images. Use the src attribute to link to an image. <img src="www.apimages.com/fHE6Sk.jpg">
<div> Use to indicate a section of content. Divs can be changed with CSS just like any other tag. <div>
<p>Paragraph</p>
</div>
<ol> Create a numbered list. Use <li> to create items on that list. <ol>
<li>Chocolate</li>
<li>Vanilla</li>
</ol>
<ul> Create an unordered list. Format is the same as <ol>. <ul>
<li>Basketball</li>
<li>Soccer</li>
</ul>
<s> Put a line through a piece of text, to indicate it being true incorrect. this happened in <s>2017</s> 2016
<b> Defines text which will be bold. <b>Important:</b>
<hr> A horizontal line.
They are useful for seperating sections of a document.		 <hr>
<h3>Section Two</h3>
<q> Quotation tag.Most browsers will put quotation marks around the text, like this! <q>"You miss 100% of the shots you don't take. -Wayne Gretzky"
-Michael Scott</q>
<link> Used to link to an external resource, such as a CSS stylesheet. <link rel="stylesheet" type="text/css" href="stylesheet.css">
<table> Create a table, using <tr> and <td> to create table rows and data for those rows. <table>
<tr>
<td>Math</td>
<td>English</td>
</tr>
</table>
<caption> Create a caption for a table, which appears above it. Must be placed on the line after the <table> tag. <caption>Monthly Earnings</caption>
<em> Use to emphasize text. <em>Emphasized text</em>
