HTML Tags Reference

Tag Description
<!-- --> The comment tag. Put text in the middle, and it won't affect the code. Useful for reminding yourself what a section of code does. Example: <!--Second Table-->
<h1>, <h2>, etc. Header tags. The number indicates the size, with larger numbers being smaller headers.
<p> The paragraph tag, pretty self-explanatory.
<a> The hyperlink tag, use the href="" attribute to link to another page. Example: <a href="http://www.google.ca">
<title> This tag sets a title for your page, which shows up in the browser tab.
