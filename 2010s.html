The 2010s

  • 2010: Apple unveils the iPad, a new way to use iOS that feels more like a desktop computer.
  • 2011: Google releases the Chromebook, a laptop designed for browsing the web with Google Chrome.
  • 2015: Microsoft releases Windows 10, which is controversial due to several changes Microsoft made.
  • 2016: The first reprogrammable quantum computer is invented, which can be used for complex simulations and rapid calculations.
  • 2017: DARPA starts developing a program called "Molecular Informatics", which uses molecules as computers.
